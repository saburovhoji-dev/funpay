# FunPay Bot — деплой на Railway

## Шаг 1 — Загрузите код на GitHub

Создайте новый **приватный** репозиторий на github.com и загрузите все файлы проекта:

```
funpay_bot/
├── main.py
├── database.py
├── funpay_client.py
├── tg_bot.py
├── requirements.txt
└── railway.toml
```

---

## Шаг 2 — Создайте проект на Railway

1. Зайдите на [railway.app](https://railway.app) → **New Project**
2. Выберите **Deploy from GitHub repo**
3. Подключите ваш аккаунт GitHub и выберите созданный репозиторий
4. Railway автоматически обнаружит `railway.toml` и `requirements.txt`

---

## Шаг 3 — Добавьте Volume (постоянное хранилище для БД)

> ⚠️ Без этого шага база данных будет стираться при каждом рестарте!

1. В вашем проекте на Railway нажмите **+ Add Service → Volume**
2. Укажите **Mount Path**: `/data`
3. Нажмите **Create**

---

## Шаг 4 — Задайте переменные окружения

В вашем сервисе на Railway перейдите в **Variables** и добавьте:

| Переменная             | Значение                          | Описание                          |
|------------------------|-----------------------------------|-----------------------------------|
| `TELEGRAM_BOT_TOKEN`   | `1234567890:ABC...`               | Токен от @BotFather               |
| `OWNER_TELEGRAM_ID`    | `123456789`                       | Ваш числовой Telegram ID          |
| `DB_PATH`              | `/data/funpay_bot.db`             | Путь к БД на Volume               |

Опциональные переменные:

| Переменная                 | По умолчанию | Описание                      |
|----------------------------|--------------|-------------------------------|
| `ORDER_CHECK_INTERVAL_SEC` | `10`         | Интервал проверки заказов (с) |
| `LIFT_INTERVAL_SEC`        | `7200`       | Интервал автоподнятия (с)     |

---

## Шаг 5 — Деплой

После добавления переменных Railway автоматически задеплоит проект.
Следите за логами во вкладке **Deployments**.

Успешный старт выглядит так:
```
2024-01-01 12:00:00 [INFO] root: База данных инициализирована.
2024-01-01 12:00:00 [INFO] root: Запущен мониторинг заказов.
2024-01-01 12:00:00 [INFO] root: Запущен цикл автоподнятия лотов.
2024-01-01 12:00:00 [INFO] root: Запускаю polling...
```

В Telegram придёт сообщение: **🚀 FunPay Bot запущен на Railway**

---

## Шаг 6 — Первоначальная настройка через бота

1. Откройте бота → `/start`
2. Нажмите **🔑 Установить golden_key**
3. Получите `golden_key` из браузера:
   - Зайдите на funpay.com в браузере
   - DevTools (F12) → Application → Cookies → `funpay.com` → `golden_key`
   - Скопируйте значение
4. Отправьте ключ боту — он проверит валидность на FunPay

---

## Добавление товаров для автовыдачи

1. **📦 Товары → ➕ Добавить товар**
2. Введите **FunPay ID лота** — число из URL: `funpay.com/lots/offer?id=`**`123456`**
3. Введите категорию (для вашего удобства)
4. Введите **текст автовыдачи** — именно это получит покупатель в чат
